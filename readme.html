JavaBot Documentation



Adding a function to be called during an event:
bot.events.add(sect, priority, flags, sub1, func1, sub2, func2 ...)
Arguments
sect
one of the sections listed in "Events" documentation
priority
When the functions listed should be called in relation to other functions
Default is 0. Set negative if function should be called earlier, positive if later.
flags
1
Default function. If there are other functions for this event, will not be called.
-1
Event calls are stopped after this function is called
Raising an event:
bot.events.call(sect, sub, *args, **kwargs)

*args and **kwargs are passed to each function that is during the event.
